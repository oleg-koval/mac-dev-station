## [0.1.0] - 2024-01-01

### Added
- Initial release with 13 idempotent setup phases
- Karabiner Hyper key configuration
- AeroSpace tiling WM setup with workspace routing
- Hammerspoon display auto-detection
- kitty terminal config with fzf project switcher
- zsh + starship + modern CLI aliases
- Homebrew tap and formula (`oleg-koval/tap`)
- GoReleaser release automation for darwin arm64/amd64
- `install.sh` for curl-install without Homebrew
