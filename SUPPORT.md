# Support

## Getting help

- **Bug reports** — [open an issue](https://github.com/oleg-koval/mac-dev-station/issues/new?template=bug_report.md)
- **Feature requests** — [open an issue](https://github.com/oleg-koval/mac-dev-station/issues/new?template=feature_request.md)
- **Questions** — [GitHub Discussions](https://github.com/oleg-koval/mac-dev-station/discussions)

## Support expectations

This is a personal open-source project maintained in spare time.

- Issues are reviewed on a best-effort basis.
- PRs with tests and clear descriptions are most likely to be merged quickly.
- No SLA or guaranteed response time.

## Before opening an issue

1. Check existing [issues](https://github.com/oleg-koval/mac-dev-station/issues) and [discussions](https://github.com/oleg-koval/mac-dev-station/discussions).
2. Run `mac-dev-station --version` and include the output.
3. Include your macOS version and chip (Apple Silicon / Intel).
4. If a phase fails, include the full terminal output of that phase.
